# Welcome to [Slidev](https://github.com/slidevjs/slidev)!

The main deck lives in `learning_progress.slides.md` and all scripts point to it.

To start the slide show:

- `npm install`
- `npm run dev`
- visit <http://localhost:3030>

Place any JSON or other static assets under `public/` (for example `public/traces/...`) so they are served without bundling and can be fetched via `/traces/file.json`.

Learn more about Slidev at the [documentation](https://sli.dev/).
